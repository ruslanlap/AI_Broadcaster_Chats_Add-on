Welcome to AI Chat Broadcaster

Thank you for installing AI Chat Broadcaster! This extension allows you to simultaneously send the same message to multiple open AI chat tabs, saving your time and effort.

Note: For this extension to work, you need to have tabs open with supported AI chat platforms.

Supported AI Platforms

  • OpenAI ChatGPT (chat.openai.com)
  • Anthropic Claude (claude.ai)
  • Grok
  • Mistral
  • DeepSeek

How to Use

1
Open AI Chats in Separate Tabs

Log into your accounts on supported platforms (ChatGPT, Claude, Grok, Mistral, DeepSeek) and open them in separate browser tabs.

2
Click the Extension Icon

Click on the AI Chat Broadcaster icon in the Firefox toolbar. In the popup, you will see a list of detected open chat tabs.

3
Enter Your Message

Type the text you want to send to all chats in the text field.

4
Select Chats and Send

Check the chats you want to send your message to, and click the "Send to all selected chats" button.

Tips and Recommendations

  • Make sure you are logged into all necessary AI chat platforms before using the extension.
  • If your message doesn't send automatically, check if the text input field is visible on the chat page.
  • For best results, we recommend sending a test message first.
  • Keep in mind that AI chat interfaces may change, which can occasionally affect the extension's functionality.
